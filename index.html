Angular with Bootstrap

Once you have your project ready you have to install boostrap and jquery

npm install bootstrap
npm install jquery

You can see the versions you have installed in the package.json file.

{
  ..
  "dependencies": {
     ...
    "bootstrap": "^4.4.1",
    "jquery": "^3.4.1",
    ...
  },
  ...
}

Then, you have to configure angular.json file.

{
  ...
  "projects": {
      ...
      "architect": {
        "build": {
          ...
          "options": {
            ...
            "styles": [
              ...
              "node_modules/bootstrap/dist/css/bootstrap.min.css"
            ],
            "scripts": [
              "node_modules/jquery/dist/jquery.min.js",
              "node_modules/bootstrap/dist/js/bootstrap.min.js"
            ]

After this, you can use Bootstrap normally because the following files have been installed.

bootstrap.min.css
jquery.min.js
bootstrap.min.js
